# Monica V4 API Scope: Monica Companion V1

_This document lists the specific Monica v4 API endpoints/operations needed for the first release. To be populated with request/response contracts and test expectations._

---

## Contacts

- List contacts
- Search contacts by name
- Get one contact by ID
- Create a new contact
- Update contact basic details (name, birthday, phone, email, address)
- Update contact work/career details

## Reminders

- List reminders (including upcoming dates)

## Notes

- List notes
- List notes for a specific contact
- Get one note by ID
- Create a note
- Update a note
- Delete a note

## Activities

- List activities for a contact
- Create an activity (log a call, meeting, etc.)
