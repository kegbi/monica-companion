# Product Definition Lite

## Project Name

Telegram-First Personal Assistant (Monica-Backed)

## Vision

Create a reliable, command-first assistant in Telegram that helps users manage relationship data, notes, and reminders through Monica-backed workflows, while staying modular for future connector expansion.

## Target Audience

Telegram users who want a fast, private, and simple way to maintain relationship context and reminders without daily use of a full CRM interface.

## Core Features

- Private Telegram command interface.
- Monica-backed contacts operations.
- Monica-backed notes operations.
- Reminder lookup and scheduled digests.
- Voice-to-text command intake.
- Streaming responses for long-running requests.
